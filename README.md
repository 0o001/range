# range
create an array range from number

```js
Array.range(5)
//output: [ 0, 1, 2, 3, 4 ]

Array.range(5, 10)
//output: [ 5, 6, 7, 8, 9 ]

Array.range(5, 10, 2)
//output: [ 5, 7 ]

Array.range(-50, -10, 2)
/*output:
[
  -50, -48, -46, -44, -42,
  -40, -38, -36, -34, -32,
  -30, -28, -26, -24, -22,
  -20, -18, -16, -14, -12
]*/
```
